# Retail Sales Dashboard

## Project Objective

Analyze retail sales data to understand customer purchasing behavior, product performance, and revenue trends.

---

## Tools Used

Excel
SQL
Power BI

---

## Dataset

Retail Sales Dataset

---

## Data Cleaning (Excel)

Steps performed:

• Removed duplicate records
• Handled missing values
• Standardized date format
• Converted numerical columns correctly
• Created calculated columns for deeper analysis

---

## SQL Analysis

Example Query:

SELECT category, SUM(sales) AS total_sales
FROM retail_sales
GROUP BY category
ORDER BY total_sales DESC;

---

## Power BI Dashboard

Dashboard includes:

• Total Revenue KPI
• Sales by Category
• Monthly Revenue Trend
• Customer Purchase Patterns
• Top Products Analysis

---

## Key Insights

• Certain product categories generate higher revenue
• Sales show consistent monthly growth trends
• Customer purchasing patterns vary across categories
